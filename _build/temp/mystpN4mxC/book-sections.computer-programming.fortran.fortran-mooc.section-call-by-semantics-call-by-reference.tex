\subparagraph{Section Call By Semantics: Call By Reference}

Adapted from: \href{https://github.com/gjbex/Fortran-MOOC/tree/master/source\_code/call\_by\_semantics}{https://github.com/gjbex/Fortran-MOOC/tree/master/source\_code/call\_by\_semantics}

\subparagraph{This program demonstrates calling by reference in Fortran.}

\begin{verbatim}
program call_by_reference
    implicit none
    integer :: m
    m = 5
    print *, m
    call increment(m)
    print *, m

contains

    subroutine increment(n)
        implicit none
        integer, intent(inout) :: n

        n = n + 1
    end subroutine increment

end program call_by_reference
\end{verbatim}

The above program is compiled and run using Fortran Package Manager (fpm):

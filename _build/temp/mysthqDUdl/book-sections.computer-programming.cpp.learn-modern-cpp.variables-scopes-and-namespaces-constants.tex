\subparagraph{Variables, Scopes and Namespaces: Constants}

Adapted from: ``Learn Modern C++'' by cpptutor: \href{https://learnmoderncpp.com/variables-scopes-and-namespaces/}{Learn Modern C++: Variables, Scopes and Namespaces}

\subparagraph{Program that Demonstrates Constants}

\begin{verbatim}
// 02 -constants.cpp : introducing the const keyword
 
#include <print>
using namespace std;
 
const double PI = 3.14159265358979;
 
int main() {
    auto const APPROX_E = 3;
    println("pi is almost exactly {}, while e is approximately {}",
        PI, APPROX_E);
}
\end{verbatim}

\subparagraph{Explanation of the Above Code}

This C++ code demonstrates the use of the \texttt{const} keyword to define constants and the use of the \texttt{println} function for formatted output.

\subparagraph{Code Explanation:}

\begin{enumerate}
\item \textbf{Header Inclusion}:

\begin{verbatim}
#include <print>
\end{verbatim}

\begin{itemize}
\item The \texttt{\textless print\textgreater } header is part of the C++23 standard library. It provides the \texttt{println} function, which is used for formatted output similar to Python's \texttt{print} function.
\end{itemize}


\item \textbf{Namespace}:

\begin{verbatim}
using namespace std;
\end{verbatim}

\begin{itemize}
\item This allows the program to use standard library features (like \texttt{println}) without needing to prefix them with \texttt{std::}.
\end{itemize}


\item \textbf{Constant Definition}:

\begin{verbatim}
const double PI = 3.14159265358979;
\end{verbatim}

\begin{itemize}
\item \texttt{PI} is defined as a constant of type \texttt{double}. The \texttt{const} keyword ensures that its value cannot be modified after initialization.
\end{itemize}


\item \textbf{Main Function}:

\begin{verbatim}
int main() {
    auto const APPROX_E = 3;
    println("pi is almost exactly {}, while e is approximately {}", PI, APPROX_E);
}
\end{verbatim}

\begin{itemize}
\item Inside the \texttt{main} function:

\begin{itemize}
\item \texttt{APPROX\_E} is declared as a constant using \texttt{auto const}. The \texttt{auto} keyword deduces its type as \texttt{int} based on the assigned value \texttt{3}.
\item The \texttt{println} function is used to print a formatted string. The placeholders \texttt{\{\}} are replaced with the values of \texttt{PI} and \texttt{APPROX\_E}.
\end{itemize}
\end{itemize}
\end{enumerate}

\subparagraph{Output:}

When executed, the program will output:

\begin{verbatim}
pi is almost exactly 3.14159265358979, while e is approximately 3
\end{verbatim}

\subparagraph{Key Concepts:}

\begin{itemize}
\item \textbf{\texttt{const} Keyword}: Used to define immutable variables.
\item \textbf{\texttt{auto} Keyword}: Automatically deduces the type of a variable.
\item \textbf{Formatted Output}: The \texttt{println} function simplifies printing formatted strings in C++23.
\end{itemize}

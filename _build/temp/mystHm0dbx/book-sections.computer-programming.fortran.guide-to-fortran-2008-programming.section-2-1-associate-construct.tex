\subparagraph{Section 2.1: Statement Blocks}

Adapted from: ``Guide to Fortran 2008 Programming'' by Walter S. Brainerd (Springer 2015)

\subparagraph{Program to demonstrate the \textbf{block} construct in Fortran}

\begin{verbatim}
program block_test

    implicit none
    real, parameter :: x = 1.1

    block
        integer :: x
        do x = 1, 3
            print *, x
        end do
    end block

    print *, x

end program block_test
\end{verbatim}

The above program is compiled and run using Fortran Package Manager (fpm):

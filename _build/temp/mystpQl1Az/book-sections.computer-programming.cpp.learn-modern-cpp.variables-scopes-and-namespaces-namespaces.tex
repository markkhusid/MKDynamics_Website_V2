\subparagraph{Variables, Scopes and Namespaces: Namespaces}

Adapted from: ``Learn Modern C++'' by cpptutor: \href{https://learnmoderncpp.com/variables-scopes-and-namespaces/}{Learn Modern C++: Variables, Scopes and Namespaces}

\subparagraph{Program that Demonstrates using Namespaces}

\begin{verbatim}
// 02 -height.cpp : define the same variable name in two different namespaces
 
#include <print>
using namespace std;
 
namespace Wonderland {
    auto alice_height_m{ 0.15 };
}
 
namespace VictorianEngland {
    auto alice_height_m{ 0.9 };
}
 
int main() {
    println("Alice\'s height varies between {}m and {}m",
        Wonderland::alice_height_m,
        VictorianEngland::alice_height_m);
}
\end{verbatim}

\subparagraph{Explanation of the Above Code}

This C++ code demonstrates the use of \textbf{namespaces} to define variables with the same name in different contexts, avoiding naming conflicts.

\subparagraph{Code Breakdown:}

\begin{verbatim}
#include <print>
using namespace std;
\end{verbatim}

\begin{itemize}
\item The \texttt{\textless print\textgreater } header is used for formatted output (similar to \texttt{std::format} in C++20). It allows the use of \texttt{println} for printing formatted strings.
\item \texttt{using namespace std;} allows direct access to standard library functions without prefixing them with \texttt{std::}.
\end{itemize}

\begin{verbatim}
namespace Wonderland {
    auto alice_height_m{ 0.15 };
}
\end{verbatim}

\begin{itemize}
\item A namespace \texttt{Wonderland} is defined.
\item Inside this namespace, a variable \texttt{alice\_height\_m} is declared and initialized with the value \texttt{0.15} (meters).
\end{itemize}

\begin{verbatim}
namespace VictorianEngland {
    auto alice_height_m{ 0.9 };
}
\end{verbatim}

\begin{itemize}
\item Another namespace \texttt{VictorianEngland} is defined.
\item Inside this namespace, a variable \texttt{alice\_height\_m} is declared and initialized with the value \texttt{0.9} (meters).
\end{itemize}

\subparagraph{Purpose of Namespaces:}

\begin{itemize}
\item Namespaces allow variables with the same name (\texttt{alice\_height\_m}) to coexist without conflict because they are scoped to their respective namespaces (\texttt{Wonderland} and \texttt{VictorianEngland}).
\end{itemize}

\begin{verbatim}
int main() {
    println("Alice's height varies between {}m and {}m",
        Wonderland::alice_height_m,
        VictorianEngland::alice_height_m);
}
\end{verbatim}

\begin{itemize}
\item The \texttt{main} function prints a formatted string using the \texttt{println} function.
\item The variables \texttt{Wonderland::alice\_height\_m} and \texttt{VictorianEngland::alice\_height\_m} are accessed using the \textbf{namespace resolution operator} (\texttt{::}).
\item The output will display the values of \texttt{alice\_height\_m} from both namespaces.
\end{itemize}

\subparagraph{Output:}

The program will produce the following output:

\begin{verbatim}
Alice's height varies between 0.15m and 0.9m
\end{verbatim}

\subparagraph{Key Concepts:}

\begin{enumerate}
\item \textbf{Namespaces}: Namespaces are used to organize code and prevent naming conflicts by grouping related variables, functions, or classes.
\item \textbf{Namespace Resolution Operator (\texttt{::})}: This operator is used to access members of a specific namespace.
\item \textbf{Variable Scoping}: The same variable name can exist in different namespaces without conflict.
\end{enumerate}

This program demonstrates how namespaces can be used to manage naming conflicts in C++ effectively.
